## Movie Recommendation System
This web application is a Movie Recommendation System built using Streamlit. Users can select a movie and receive recommendations for similar or related movies.

#### Features
Simple movie selection: Choose a movie from a dropdown list.

Instant recommendations: Get recommended movies based on your selection.

Intuitive UI: Clean interface powered by Streamlit.

#### How to Use
Access the app: <a href="https://movie-recommendation-system-bkmdd2ztqojbbhzvjwpdlt.streamlit.app/">view app </a>

Choose a movie: Use the dropdown or combobox to select a movie title (e.g., "(500) Days of Summer").

View recommendations: Click the "Recommend" button and see a list of suggested movies.

Streamlit for web interface
